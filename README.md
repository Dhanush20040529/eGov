eGov Website Clone – Technical Task
Project Overview

This project is a frontend technical task to recreate the eGov Foundation website landing page using HTML, CSS, Bootstrap 5, and JavaScript.

The goal was to replicate the design as closely as possible while maintaining:

Responsive layout
Clean code structure
Bootstrap + custom CSS combination
Mobile-first responsiveness
Smooth UI interactions
Scroll animations using AOS

The project includes multiple sections such as:

Top Bar + Sticky Navbar
Hero Section
Statistics Section
Problem Section
Sustainable Change Section
DIGIT Platform Section
Latest at eGov Section
Footer Section

The HTML structure is built in index.html and the styling is handled in style.css .
